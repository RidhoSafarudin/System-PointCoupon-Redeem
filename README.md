# Project System-Coupon-Redeem

## Project info

## Deskripsi
Proyek ini adalah platform berbasis web untuk menukarkan kupon (Redeem Coupon) guna mendapatkan berbagai hadiah menarik. Sistem ini dirancang untuk memvalidasi kode kupon dan memberikan hadiah sesuai dengan ketersediaan atau jenis kupon yang digunakan. Proyek ini dibuat dengan bantuan Lovable.

## How to run
Sebelum memulai, pastikan Anda telah menginstal Node.js & npm di perangkat Anda.
Cara Install jika belum
(https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

**ikuti langkah2 berikut jika sudah menginstall Node.js dan npm:

```sh
# Step 1: Clone repository menggunakan URL Git project.
git clone https://github.com/RidhoSafarudin/System-PointCoupon-Redeem.git

# Step 2: Masuk ke direktori project.
cd System-PointCoupon-Redeem

# Step 3: Install dependency yang diperlukan.
npm i

# Step 4: Jalankan development server dengan auto-reload dan preview instan.
npm run dev
```

Project ini dibuat dengan:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
